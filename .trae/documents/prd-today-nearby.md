## 1. 产品概述
在“今日”页新增「附近」板块，基于你的定位展示周边咖啡/饮品店列表。
当定位权限不可用时提供清晰的授权引导与可用的降级体验，保证内容不断层。

## 2. 核心功能

### 2.1 功能模块
本次需求涉及的页面包括：
1. **今日页**：附近板块（定位权限处理、周边店铺列表、降级入口与状态展示）。

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---|---|---|
| 今日页 | 附近-权限与状态 | 识别定位状态（未请求/允许/拒绝/失败）。在首次进入或点击“开启定位”时触发系统定位授权；在不同状态下展示对应文案、按钮与降级入口。 |
| 今日页 | 附近-店铺列表 | 在允许定位时，展示周边咖啡/饮品店列表（店名、距离、地址/商圈、品类标签）。默认按距离排序，支持手动刷新。 |
| 今日页 | 附近-店铺操作 | 点击店铺条目进入外部地图（新窗口/新标签）进行导航；支持一键复制地址（可选：复制店名+地址）。 |
| 今日页 | 附近-降级方案 | 当定位被拒绝或不可用时：提供“选择城市/地区”入口并展示“该地区热门饮品店”列表；当网络/API异常时：展示空态+重试，并允许继续浏览今日页其他内容。 |

## 3. 核心流程

### 3.1 附近板块（允许定位）
1. 你打开“今日”页，看到「附近」卡片骨架屏。
2. 若你尚未授予定位权限，点击“开启定位”。
3. 授权成功后，系统获取当前位置并拉取周边店铺列表。
4. 你可以点击某一店铺，跳转外部地图进行导航；也可以下拉/点击刷新。

### 3.2 附近板块（拒绝定位/不可用的降级）
1. 若你拒绝定位权限，「附近」展示解释文案与“去系统设置开启/选择城市”按钮。
2. 你选择城市/地区后，系统展示该地区热门饮品店列表（不展示距离，仅展示地址/商圈与品类）。
3. 你仍可点击店铺条目打开外部地图。

### 3.3 异常处理
1. 若定位获取失败（超时/无GPS/浏览器不支持），展示“定位不可用”提示与“选择城市/重试”。
2. 若周边API请求失败，展示错误态并允许重试；不阻塞今日页其他模块。
