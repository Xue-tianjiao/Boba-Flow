# 页面设计：搜索与灵感发现中心（桌面优先）

## 全局设计规范（适用于全部页面）
### Layout
- 桌面优先：内容最大宽度 1200px，左右留白居中；主栅格采用 CSS Grid（12 列），词云区与结果区为双栏布局（8/4）。
- 响应式：≤1024px 改为上下结构（词云在上、结果在下）；≤768px 搜索框占满宽度，详情使用全屏抽屉。

### Meta Information
- Title：DrinkMe｜搜索与灵感发现
- Description：全局模糊搜索与动态词云，发现品牌新品卖点。
- OG：og:title、og:description、og:type=website

### Global Styles（Design Tokens）
- 背景：#0B0F14（深色） / 卡片：#121824
- 主色：#5EEAD4（青绿）
- 强调：#A78BFA（紫）
- 文本：#E5E7EB；次级文本：#94A3B8
- 字体：系统字体 + 等宽用于数据（如更新时间）
- 圆角：12px；阴影：轻量外发光（hover）
- 按钮：
  - Primary：主色填充 + 深色文字；hover 亮度 +8%
  - Secondary：描边按钮（主色描边）；hover 背景半透明

## 页面 1：搜索与灵感发现（今日）页

### Page Structure
- 顶部导航（固定）
- 主内容双栏：左侧“漂浮词云舞台”，右侧“聚合结果/快捷筛选”
- 底部：数据更新时间与“数据来源说明”入口

### Sections & Components
1) 顶部导航（Top Nav）
- 左：产品名/Logo
- 中：全局搜索框（核心）
- 右：刷新控件组（仅重排 / 换一批）+ 清空筛选

2) 全局搜索框（Fuzzy Search Bar）
- 输入框：支持模糊匹配；输入时展示下拉联想（最多 8 条）
- 联想分组：词条（Term）/ 品牌（Brand）/ 卖点（Selling Point）
- 交互：
  - Enter：进入“聚合结果视图”（仍在同页右栏）
  - Esc：清空输入并返回词云浏览态
  - 高亮：匹配部分加粗/主色下划线

3) 词云舞台（Floating Word Cloud Stage）
- 容器：左栏卡片，内部为 Canvas 或 SVG（推荐 Canvas + 绝对定位标签）
- 词条样式：
  - 字号：根据 weight 映射（例如 14px-44px）
  - 颜色：默认次级文本；hover 主色；选中态强调色
- 动态漂浮：
  - 基础布局：词云算法生成初始不重叠坐标
  - 漂浮动画：每个词条在 small radius 内做缓慢位移（requestAnimationFrame），并带轻微视差
  - 防遮挡：检测边界与近距离时进行微小回弹
- 交互：
  - Hover：浮层 tooltip（品牌/新品/卖点摘要、更新时间）
  - Click：选中词条并打开右栏详情/结果

4) 刷新重排（Refresh/Relayout Controls）
- “仅重排”：保留同一批 terms，重新计算布局 + 重新随机漂浮轨迹
- “换一批”：请求后端生成/拉取新 terms，替换并重排
- Loading：按钮进入旋转态，词云舞台显示骨架/轻提示

5) 右栏：聚合结果区（Results Panel）
- 默认态（未搜索/未选词条）：展示“今日推荐探索路径”（例如：按品牌/按卖点）
- 搜索/选中态：
  - 顶部：当前筛选 chips（可逐个关闭）
  - 排序：热度 / 最新
  - 列表：关联饮品卡片（名称、品牌、关键卖点 1-2 条、来源/更新时间）
  - 空态：给出“换一批/清空/尝试其它关键词”

## 页面 2：词条/饮品详情页（或右侧抽屉）

### Layout
- 桌面：右侧抽屉（宽 420px）覆盖在今日页上，不丢上下文
- 移动端：全屏抽屉

### Sections & Components
1) 头部
- 标题：displayText（品牌+新品+卖点摘要）
- 次信息：更新时间（ISO 转友好时间）
- 操作：关闭、复制词条

2) 卖点列表
- 以要点列表呈现 sellingPoints（每条 ≤18 字为佳）

3) 关联饮品卡片
- 卡片：饮品名、品牌、卖点标签、来源链接
- 点击卡片：可在同抽屉内切换到“饮品详情”子视图（若产品已存在该概念）

4) 数据来源
- 展示 sourceUrls（域名 + 标题简写），支持外链打开
